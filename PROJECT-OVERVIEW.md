# Decants-ni-bro: Perfume Decanting Web Application

## App Description
Decants-ni-bro is a modern web application designed for selling perfume decants. It features a public-facing catalog for customers to browse available fragrances and an administrative dashboard for managing inventory, tracking sales, and analyzing statistics.

## Target Users
- **Customers**: Individuals looking to explore and purchase high-quality perfume decants.
- **Administrators**: The business owners managing product listings, stock levels, and reviewing sales performance.

## Key Features
- **Public Catalog**: A responsive, visually appealing display of available perfume decants with filtering and search capabilities.
- **Admin Dashboard**: A secure area for administrators to manage the application.
- **Inventory Management**: Tools to add, update, and remove products, monitor stock levels, and handle restocks.
- **Sales Recording**: A system to log and track completed transactions.
- **Statistics & Analytics**: Visual insights into sales trends, popular products, and revenue tracking.
- **Authentication**: Secure login system with route protection for administrative areas.

## Tech Stack
- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Database**: PostgreSQL (managed via Prisma ORM)
- **Authentication**: NextAuth.js (Auth.js) or Supabase Auth
- **Deployment**: Vercel
